# class-cookbook
Our shared class project. Everyone adds ONE recipe.

## How to contribute
1. Get the latest:        git pull
2. Make your own branch:  git switch -c recipe/<your-name>
3. Add a file named:      recipes/<your-name>.txt
   (copy the layout from recipes/EXAMPLE.txt)
4. Send it up:
      git add .
      git commit -m "Add <your-name>'s recipe"
      git push -u origin recipe/<your-name>
5. Open a Pull Request here on GitHub.

⚠️ Add your OWN file only. Don't edit anyone else's file or this README.
