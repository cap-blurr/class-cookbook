cd# class-cookbook
Our shared class project. Everyone adds ONE recipe.

## How to contribute
1. Get the project (first time only):
      git clone https://github.com/cap-blurr/class-cookbook.git
      cd class-cookbook
2. Get the latest:        git pull
3. Make your own branch:  git switch -c recipe/<your-name>
4. Add a file named:      recipes/<your-name>.txt
   (copy the layout from recipes/EXAMPLE.txt)
5. Send it up:
      git add .
      git commit -m "Add <your-name>'s recipe"
      git push -u origin recipe/<your-name>
6. Open a Pull Request here on GitHub.

⚠️ Add your OWN file only. Don't edit anyone else's file or this README.
